# Design Backlog

Dispatch source for `/cmux-dispatch`. One Kanban board rendered in ten distinct visual styles; the same realistic board content is owned per entry. Order is user-adjusted (glassmorphism first). See `docs/catalog-roadmap.md` for the full direction.

- [x] `kanban-glassmorphism` — frosted translucent panels over a soft low-chroma mist wash, backdrop blur, cool-gray hairlines, calm dusty-slate accent. theme: light, density: comfortable
- [ ] `kanban-flat-material` — solid color blocks, crisp elevation shadows, bold accents. light, comfortable
- [ ] `kanban-neumorphism` — soft extruded/inset monochrome surfaces, dual shadows, tactile (AA-lifted). light, comfortable
- [ ] `kanban-claymorphism` — puffy pastel 3D shapes, marshmallow cards, soft drop shadows. light, comfortable
- [ ] `kanban-illustration` — chunky illustrated accents, stickers, friendly rounded shapes. light, spacious
- [ ] `kanban-editorial` — serif display type, generous whitespace, content-first print palette. light, spacious
- [ ] `kanban-swiss` — strict grid, austere sans, monochrome + one accent, high craft. light, comfortable
- [ ] `kanban-brutalism` — raw monospace, harsh borders, black/white + one accent, exposed grid. light, compact
- [ ] `kanban-dark-neon` — near-black canvas, glowing neon accents, monospace, terminal vibe. dark, compact
- [ ] `kanban-skeuomorphic` — realistic textures and depth, bevels, physical affordances. light, comfortable
